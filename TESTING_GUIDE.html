Supplier Risk Scorecard

150 suppliers scored across 6 risk dimensions, classified into High / Medium / Low tiers with spend concentration analysis.

Setup — Run Once

1

Install packages

pip3 install -r requirements.txt

Only needed the first time. Same packages work for all projects.

How to Run

2

Go to the folder

cd ~/Desktop/Jay_Portfolio_Projects/supplier-risk-scorecard
3

Run the project

python3 main.py
4

Open the dashboard

open outputs/supplier_dashboard.html

What to Check

  • All 3 risk tiers have suppliers
  • Risk dimension chart shows 6 different scores
  • Top 10 risky suppliers shown with names
  • SQLite Database: After running, check outputs/data.db — all data is stored in SQLite and analyzed with SQL queries.
